Define internet backbone in your own words. The internet backbone is a series of routers that connect to fibre-optic cables. These cables are the main way by which data is transferred around the world. The internet backbone connects the continents via underwater cables, and is built with backup routes in the case of a cable failure. What is the difference between clients and servers on the internet? A client is the side of an exchange of data on the internet that recieves or requests data from a server. The server is the side which sends data to a client. The two can be compared to a cashier (server) and customer (client).

Describe the Internet Protocol Suite (TCP/IP) model of networking, in general terms, outlining how this protocol provides the data transfer mechanism required to establish client/server relationships.

The Internet Protocol Suite describes a set of protocols, rules that computers must follow in order to ensure that the data they transfer is correct, to the right place, and to the right time. IP ensures data gets to the right place, TCP ensures it is in the correct order and none is missing.

Describe and illustrate the development of the Web in general terms using the Web 1.0, Web 2.0, and Web 3.0 generational paradigms.

    The next framework of the internet arrived; the Web 2.0. Included were ease of access to more parts of the internet, social interaction and commenting, content that responds to user input, and integrated APIs. The first blogging and social media sites started appearing.

Web 1.0 focuses on read-only websites and very static web design. Web 2.0 focuses on accessibility, social interaction, and data exchange. Web 3.0 is purported to improve automation and customization, while focusing on decentralization.
